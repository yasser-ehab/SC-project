EdgeConvert is a Java application that can read in different file types, read their schema, and then output the schema into the format of the user's choosing. EdgeConvert is also very flexible and allows the addition of new formats. Please see the Add Features section for more info.

Get started

  • Open Files - Open a file to load the schema

  • Define Tables - Edit tables and fields

  • Define Relationships - Edit relationships amongst the tables

  • Save - Output to the file type of your choice

  • Add Features - Adding features to EdgeConvert

How to Open Files

1. Opening the file

  • Select "File" from the top menu bar"
  • To open one of the file types in the plug-in directory click "Open"
    • Then choose one of the file types from the child menu
  • Or click "Open Save File" to open a file saved by edge convert.

2. Select and open

  • You will be prompted to select the file of the file type you selected.
  • Select file and click "Open"
  • To cancel click "Cancel"
  • After opening the file you will be ready to define your tables.

Defining Tables

1. Selecting tables and fields

  • Once you have opening a file, you can define your fields in your table
  • To show fields, select a table under "All Tables"
  • To begin defining a field select one from under the "Field List"
    • You may also use the up and down arrows to the right to move through the list
  • You may then begin defining the field's properties
    • By selecting type from the radio buttons
    • By defining if it is a primary key, allows a null value, or should increment automatically

2. Setting lengths and defaults

  • Set the length of varchar and char types by clicking on "Set Char Length"
    • Prompt shown below
  • Set the default value of a type by clicking on "Set Default Value"

Defining Relationships

1. Defining relationships

  • To begin defining relations select "Define Relations" on the table GUI
  • To define a relation:
    • Select a table under "Tables With Relations"
    • Select a table under "Related Tables"
    • Select the fields you would like to bind or unbind a relationship from "Fields in Tables with Relations" and "Fields in Related Tables"
    • Once your GUI appears like the image to the left, select "Bind/Unbind Relation" to bind or unbind the relation.
  • To get back to defining tables select "Define Tables"

2. Notes on bind/unbind

  • If the data types do not match you will not be able to bind the relation
  • If the selected table does not have any foreign keys, all fields will appear in the "Fields in Tables with Relations" list box.
  • If two fields are bound or unbound, you will be notified with a prompt.

Saving Files

1. Saving a file

  • To save a file select "Create DDL" in either the Define Tables or Define Relations view.
  • You will then be prompted to select the file type you'd like to save the file in.
  • Then select the directory and declare the file name for your file without the extension.
  • Finally you will click "Save" to save the file.
  • You will be prompted on the file saving success or failure.

EdgeConvert is a very flexible program and allows developers to add their own file types to the application so they may parse and/or save files in the file types they desire.

1. Selecting your own plugin directory

  • You may select your own directory of plugins by select "Options" from the top menu and then selecting "Set Plugin Directory"
  • This will then be followed by a dialog where you will be able to select the directory of plug-ins.
  • This choice is better suited for developers who would like to temporarily use plug-ins not included with the application.

2. Other options

  • You may also add plug-ins on a more permanent basis to the application.
  • To do so add your plug-ins in the plugin directory in the install location of the applications jar.
